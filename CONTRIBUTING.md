# Adding an Entry

1. Create a markdown file in the relevant category folder: `category/project-name.md`
2. Use this template:

```md
# Project Name

**Author:** ...
**Source:** [link text](url)
**Category:** ...
**Added:** YYYY-MM-DD

## What it is

(1-2 sentences)

## Why it's interesting

- bullet points

## Tags

`tag1` `tag2` `tag3`

## Notes

(optional: setup, porting notes, caveats)
```

3. Add a row to the index table in `README.md`

## Categories

- `shaders/` — GLSL / HLSL / shader art
- `webgl/` — WebGL, Three.js, Babylon.js
- `canvas2d/` — HTML5 Canvas 2D
- `generative/` — Procedural / generative art (p5.js, etc.)
- `data-viz/` — Animated data visualizations
- `tools/` — Graphics tools, utilities, libraries
